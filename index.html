SECTION 01

2nd Order System

PROBLEM STATEMENT

Given transfer function: G(s) = 12 / (5s² + 2.1s + 3)
Step input of magnitude 4. Find: % overshoot, decay ratio, rise time, settling time, period of oscillation, max Y(t), ultimate Y(t). Draw pole-zero plot.

Step 1 — Convert to Standard 2nd-Order Form

The standard form is:

G(s) = K / (τ²s² + 2ζτs + 1) Divide numerator and denominator by 3: G(s) = (12/3) / [(5/3)s² + (2.1/3)s + 1] = 4 / [1.667s² + 0.7s + 1]

Matching coefficients:

τ² = 1.667 → τ = √1.667 = 1.291 min // natural period 2ζτ = 0.7 → ζ = 0.7 / (2 × 1.291) = 0.271 // damping ratio K = 4 // steady-state gain ωₙ = 1/τ = 1/1.291 = 0.775 rad/min // natural frequency
Check: Since 0 < ζ = 0.271 < 1, the system is underdamped — it will oscillate.

Step 2 — Damped Natural Frequency

ωd = ωₙ × √(1 − ζ²) = 0.775 × √(1 − 0.271²) = 0.775 × √(1 − 0.0734) = 0.775 × √0.9266 = 0.775 × 0.9626 = 0.746 rad/min

Step 3 — All Performance Parameters

% Overshoot (OS) = exp(−πζ / √(1−ζ²)) × 100 = exp(−π × 0.271 / √0.9266) × 100 = exp(−0.8513 / 0.9626) × 100 = exp(−0.8844) × 100 = 0.4130 × 100 = 41.3 % Decay Ratio (DR) = (OS/100)² = (0.413)² = 0.1706 // or DR = exp(−2πζ/√(1−ζ²)) = e^(−1.769) = 0.171 ✓ Period of Oscillation (T) = 2π / ωd = 2π / 0.746 = 8.42 min Time of First Peak (tp) = π / ωd = π / 0.746 = 4.21 min Rise Time (tr) = (π − cos⁻¹ζ) / ωd = (π − cos⁻¹(0.271)) / 0.746 = (3.1416 − 1.297) / 0.746 = 1.844 / 0.746 = 2.47 min Settling Time (ts) = 4 / (ζ × ωₙ) // 2% criterion = 4 / (0.271 × 0.775) = 4 / 0.2100 = 19.05 min

Step 4 — Response to Step Input of Magnitude 4

Ultimate (steady-state) value: y(∞) = K × step magnitude = 4 × 4 = 16 units Maximum value of Y(t): y_max = y(∞) × (1 + OS/100) = 16 × (1 + 0.413) = 16 × 1.413 = 22.61 units (occurs at t = 4.21 min)
FINAL ANSWERS — SECTION 1 PROBLEM 1
  • τ = 1.291 min  |  ζ = 0.271  |  ωₙ = 0.775 rad/min  |  ωd = 0.746 rad/min
  • % Overshoot = 41.3%
  • Decay Ratio = 0.171
  • Rise Time = 2.47 min
  • Time to first peak = 4.21 min
  • Period = 8.42 min
  • Settling Time = 19.05 min
  • Ultimate Y(t) = 16 units
  • Maximum Y(t) = 22.61 units at t = 4.21 min

PROBLEM 2 — TWO TANKS IN SERIES (NON-INTERACTING)

Two tanks drain independently in series. Show TF = product of two 1st order systems. Comment on overshoot.

Part (a) — Why TF = Product of Two First-Order Systems

For non-interacting tanks (outflow of Tank 1 does NOT depend on level of Tank 2):

Tank 1 — mass balance: A₁ dh₁/dt = Fin − h₁/R₁ Taking Laplace: A₁s·H₁(s) = Fin(s) − H₁(s)/R₁ → H₁(s)/Fin(s) = R₁/(τ₁s + 1) where τ₁ = A₁R₁ Tank 2 — input is h₁/R₁ (outflow from Tank 1): A₂ dh₂/dt = h₁/R₁ − h₂/R₂ → H₂(s)/H₁(s) = R₂/(τ₂s + 1) where τ₂ = A₂R₂ Overall TF (output = h₂, input = Fin): G(s) = H₂(s)/Fin(s) = [H₁/Fin] × [H₂/H₁] G(s) = R₁R₂ / [(τ₁s+1)(τ₂s+1)]

Since the dynamics are independent (no hydraulic coupling), the overall TF is simply the product of two first-order TFs — each tank behaves as a separate lag element in series.

Part (b) — 2nd Order TF and Overshoot Comment

Expand: (τ₁s+1)(τ₂s+1) = τ₁τ₂s² + (τ₁+τ₂)s + 1 Compare with standard form: τ²s² + 2ζτs + 1 → τ = √(τ₁τ₂) → 2ζτ = τ₁ + τ₂ → ζ = (τ₁ + τ₂) / (2√(τ₁τ₂))
Proof that ζ ≥ 1 always (AM-GM inequality): Arithmetic Mean ≥ Geometric Mean: (τ₁ + τ₂)/2 ≥ √(τ₁τ₂) → (τ₁ + τ₂)/(2√(τ₁τ₂)) ≥ 1 → ζ ≥ 1 always!
CONCLUSION

Since ζ ≥ 1 for all non-interacting two-tank systems, the response is always overdamped or critically damped. There is NO overshoot and NO oscillation. The step response is a monotonically increasing S-shaped curve that smoothly approaches steady state.

SECTION 02

Proportional Band — PID Output

PROBLEM STATEMENT

PID flow control: PB = 50%, τI = 1 min, τD = 0.5 min, Error = 10 units, de/dt = 4 units/min, t = 2 min.
Find: controller gain, proportional output, integral output, total output.

Key Formulas

PID output: p(t) = Kc[e(t) + (1/τI)∫e dt + τD(de/dt)] + p₀ Proportional Band: PB% = 100 / Kc → Kc = 100 / PB%
Controller Gain:
Kc = 100 / PB = 100 / 50 = 2 (units of output / units of error)
Proportional Output (P-term):
P = Kc × e = 2 × 10 = 20 units
Integral Output (I-term) at t = 2 min:

Assuming error has been constant at 10 units from t=0 to t=2:

I = Kc × (1/τI) × ∫₀² e dt = 2 × (1/1) × (10 × 2) = 2 × 1 × 20 = 40 units
Derivative Output (D-term):
D = Kc × τD × (de/dt) = 2 × 0.5 × 4 = 4 units
Total Controller Output:
p(t) = P + I + D = 20 + 40 + 4 = 64 units
FINAL ANSWERS
  • Controller Gain Kc = 2
  • Proportional Output = 20 units
  • Integral Output = 40 units
  • Derivative Output = 4 units
  • Total Output p(t) = 64 units
SECTION 03

PI Controller — TF & Step Response

PROBLEM STATEMENT

Derive the transfer function of a PI controller. Estimate the dynamic response of PI controller output to a step change in error. Plot over 5 reset time intervals.

Derivation of PI Transfer Function

PI controller time-domain equation: p(t) = Kc[e(t) + (1/τI) ∫₀ᵗ e(t') dt'] + p₀ Taking Laplace transform: P(s) = Kc[E(s) + E(s)/(τI·s)] + p₀/s P(s)/E(s) = Kc[1 + 1/(τI·s)] Gc(s) = Kc · (τI·s + 1) / (τI·s) This has: → One zero at: s = −1/τI → One pole at: s = 0 (integrator — eliminates steady-state offset)

Step Response of PI Controller Output

Let step change in error = A, so E(s) = A/s:

P(s) = Gc(s) · E(s) = Kc·(τI·s + 1)/(τI·s) × A/s P(s) = Kc·A·(τI·s + 1) / (τI·s²) P(s) = KcA/s + KcA/(τI·s²) Taking inverse Laplace: p(t) = KcA + (KcA/τI)·t for t ≥ 0

This has two parts:

  • KcA → Immediate step jump (proportional action)
  • (KcA/τI)·t → Linear ramp rising with time (integral action)

PROBLEM 1 (PAGE 129) — PNEUMATIC PROPORTIONAL CONTROLLER

Temperature range: 60–100°F. Output 3 psi → 15 psi as temperature goes 71°F → 75°F. Find gain and PB.

Solution

Measurement span = 100 − 60 = 40°F Output span = 15 − 3 = 12 psi Controller Gain: Kc = ΔOutput / ΔInput = (15−3) / (75−71) = 12/4 = 3 psi/°F Proportional Band: The input span that causes full output change = 75 − 71 = 4°F PB% = (input span for full output / total measurement range) × 100 = (4 / 40) × 100 = 10% Verification: PB% = 100/Kc(normalized) → need Kc in %output/%input Kc(normalized) = (12/12 × 100) / (4/40 × 100) = 100/10 = 10 PB = 100/10 = 10% ✓
ANSWERS

Controller Gain = 3 psi/°F  |  Proportional Band = 10%

PROBLEM 3 — PID CLOSED-LOOP ANALYSIS

G_P(s) = 2/(4s+1), C(s) = 3 + 2/s + s (Kp=3, KI=2, KD=1). Find: closed-loop TF, SSE for step & ramp, rise time, settling time, effect of derivative on overshoot.

Step 1 — Controller and Open-Loop TF

C(s) = Kp + KI/s + KD·s = 3 + 2/s + s = (3s + 2 + s²) / s = (s² + 3s + 2) / s Open-loop: L(s) = C(s)·Gp(s) = [(s²+3s+2)/s] × [2/(4s+1)] L(s) = 2(s²+3s+2) / [s(4s+1)] = 2(s+1)(s+2) / [s(4s+1)]

Step 2 — Closed-Loop Transfer Function

T(s) = L(s) / [1 + L(s)] = 2(s²+3s+2)/[s(4s+1)] / {1 + 2(s²+3s+2)/[s(4s+1)]} = 2(s²+3s+2) / [s(4s+1) + 2(s²+3s+2)] Denominator: s(4s+1) + 2(s²+3s+2) = 4s² + s + 2s² + 6s + 4 = 6s² + 7s + 4 T(s) = 2(s²+3s+2) / (6s²+7s+4)

Step 3 — Steady-State Error

SSE = lim(s→0) s·E(s) = lim(s→0) s · [1-T(s)] · R(s) For STEP input R(s)=1/s: SSE = lim(s→0) [1 − T(s)] = 1 − T(0) T(0) = 2(0+0+2)/(0+0+4) = 4/4 = 1 SSE (step) = 1 − 1 = 0 ✓ (PID has integrator, always kills step offset) For RAMP input R(s)=1/s²: SSE = lim(s→0) s · [1-T(s)] / s² = lim(s→0) [1-T(s)]/s Use L'Hôpital or series expansion: T(s) ≈ 1 + (dT/ds)|₀ · s + ... From L(s): Position constant Kp_pos = lim(s→0) L(s) = ∞ (integrator in C) Velocity constant: Kv = lim(s→0) s·L(s) = lim(s→0) 2(s²+3s+2)/(4s+1) = 2×2/1 = 4 SSE (ramp) = 1/Kv = 1/4 = 0.25 units

Step 4 — Rise Time and Settling Time

Closed-loop denominator: 6s² + 7s + 4 Divide by 6: s² + (7/6)s + (4/6) = s² + 1.167s + 0.667 Standard: s² + 2ζωn·s + ωn² = 0 ωn² = 0.667 → ωn = 0.816 rad/s 2ζωn = 1.167 → ζ = 1.167/(2×0.816) = 0.714 ωd = ωn√(1−ζ²) = 0.816×√(1−0.510) = 0.816×0.700 = 0.572 rad/s Rise Time: tr = (π − cos⁻¹ζ)/ωd = (3.1416 − cos⁻¹(0.714))/0.572 = (3.1416 − 0.775)/0.572 = 2.367/0.572 = 4.14 s Settling Time: ts = 4/(ζ·ωn) = 4/(0.714×0.816) = 4/0.583 = 6.86 s

Step 5 — Effect of Derivative Term on Overshoot

Without KD (PI only, KD=0): ζ would be lower → higher overshoot With KD=1: damping is increased % Overshoot = exp(−πζ/√(1−ζ²)) × 100 = exp(−π×0.714/√(1−0.510)) × 100 = exp(−2.238/0.700) × 100 = exp(−3.197) × 100 = 4.1% The KD term adds a zero at s = −KI/KD = −2 and increases effective damping. → Derivative action REDUCES overshoot and improves transient response.
ANSWERS — PROBLEM 3
  • Closed-Loop TF: T(s) = 2(s²+3s+2) / (6s²+7s+4)
  • SSE (step) = 0 (integral action eliminates offset)
  • SSE (ramp) = 0.25 units
  • Rise Time = 4.14 s
  • Settling Time = 6.86 s
  • Overshoot with KD=1: ~4.1% (significantly reduced vs PI alone)

PROBLEM 2 — SERVO: STEP CHANGE OF 0.1 UNIT

A control system receives a setpoint step change of 0.1 unit. Find: (a) max Y and time, (b) offset, (c) period of oscillation.

Solution (Using Standard 2nd-Order Closed-Loop with P-Control)

For a typical closed-loop system shown in textbook figures (proportional controller, first-order process), the closed-loop TF is:

Assume (typical textbook figure values): Process: Gp(s) = Kp/(τs+1), Controller gain: Kc Closed-loop TF: T(s) = KcKp/(τs+1) / [1 + KcKp/(τs+1)] = KcKp / (τs + 1 + KcKp) = [KcKp/(1+KcKp)] / [τ/(1+KcKp) · s + 1] This is a 1st order CL with P-only control. If the figure shows a 2nd-order system or a P-controller with two lags, the system becomes underdamped. General approach: Given standard CL parameters (from figure): Let the CL system have: τCL, ζCL derived from figure values. For step change R = 0.1: (a) Maximum Y: y_max = (CL gain × 0.1) × (1 + OS/100) where OS = exp(−πζ/√(1−ζ²)) × 100 (b) Offset (P-only control): Offset = 0.1 × 1/(1+KcKp) = 0.1 − y(∞) y(∞) = 0.1 × KcKp/(1+KcKp) (c) Period of oscillation: T = 2π/ωd = 2π/(ωn√(1−ζ²)) Time of max: tp = π/ωd
Note: The exact numbers depend on the specific transfer function values given in the "Figure" referenced in the question. The formulas above are the exact method — substitute the Kc, Kp, τ values from your figure to get numerical answers. The method is identical to Section 1 above.

For a worked numerical example — using commonly given textbook values of Kc=1, Gp = 1/(s+1)(2s+1):

Open-loop: L(s) = 1/[(s+1)(2s+1)] CL TF: T(s) = 1 / [2s²+3s+2] → divide by 2: = 0.5 / [s² + 1.5s + 1] τ²=1 → τ=1, 2ζτ=1.5 → ζ=0.75, ωn=1, ωd=√(1−0.5625)=0.661 Step 0.1: y(∞) = 0.5×0.1 = 0.05 (gain = 0.5) Offset = 0.1−0.05 = 0.05 OS = exp(−π×0.75/0.661) = exp(−3.566) = 2.8% y_max = 0.05×1.028 = 0.051 at tp = π/0.661 = 4.75 s T = 2π/0.661 = 9.51 s
PROBLEM 3 — REGULATOR: DISTURBANCE REJECTION

Gp(s) = 1/(8s+1), Gd(s) = 1/(4s+1), Kp=2. Disturbance +10 kPa at t=0. Find: immediate effect, final pressure, SSE, speed of rejection.

Closed-Loop Response to Disturbance

For a P-controller with disturbance entering at the process input: Y(s)/D(s) = Gd(s) / [1 + Kc·Gp(s)] Substituting: Y(s)/D(s) = [1/(4s+1)] / {1 + 2×[1/(8s+1)]} = [1/(4s+1)] / [(8s+1+2)/(8s+1)] = [1/(4s+1)] × [(8s+1)/(8s+3)] Y(s)/D(s) = (8s+1) / [(4s+1)(8s+3)]

Immediate Effect (t → 0⁺, s → ∞)

lim(s→∞) Y(s)/D(s) = lim(s→∞) (8s+1)/[(4s+1)(8s+3)] = lim(s→∞) 8s / (32s²) = lim(s→∞) 1/(4s) = 0 Immediate effect = 0 kPa (Process lags prevent instantaneous pressure change)

Final Pressure After Control Action (t → ∞, s → 0)

Final value theorem: y(∞) = lim(s→0) s · Y(s) with D(s) = 10/s (step of +10 kPa) y(∞) = lim(s→0) s × [(8s+1)/((4s+1)(8s+3))] × (10/s) = lim(s→0) 10(8s+1)/[(4s+1)(8s+3)] = 10 × (1) / [(1)(3)] = 10/3 = 3.33 kPa Final pressure = Psteady + y(∞) = original + 3.33 kPa

Steady-State Error and Speed of Rejection

SSE EXISTS because P-only control cannot fully reject a disturbance. Residual pressure deviation = 3.33 kPa (offset) → To eliminate: need INTEGRAL action (PI or PID controller) Speed of disturbance rejection (dominant time constant): Closed-loop poles from denominator (4s+1)(8s+3) + (8s+1)·0 ... Actually, CL char. eq: (4s+1)(8s+1) + 2 = 0 (from 1+KcGp=0) 32s² + 12s + 1 + 2 = 0 32s² + 12s + 3 = 0 s = [−12 ± √(144−384)] / 64 → complex roots → underdamped! Or using dominant CL time constant approximation: τCL ≈ τprocess / (1 + Kc·Kp) = 8 / (1 + 2×1) = 8/3 = 2.67 min Speed: The disturbance is rejected with time constant ≈ 2.67 min
FINAL ANSWERS — REGULATOR PROBLEM 3
  • Immediate effect = 0 kPa (no instantaneous response due to lags)
  • Final pressure deviation = +3.33 kPa (residual after P-control)
  • Steady-state error EXISTS = 3.33 kPa (P-only cannot eliminate disturbance offset)
  • Speed of rejection τCL ≈ 2.67 min
  • To achieve zero offset → upgrade to PI controller
